Note on AI trajectory 2026

1.Static model as a reservoir.
  i.e. A static pretrained model can be used as a reservoir and output neurons can be trained against its output to modify it outcomes.

2. Monolithic Models -> Small core + millions of tiny cores 
  i.e. similar to transfomer hippocampus + millions of smaller transformer (cortical macro-columns)
  Macro-columns hold memory and skills

3. Memory will move out of the Core (monolithic model)
  i.e. No longer in mid to final attention weights and FFN

4. New materials will give dramatically higher energy efficiency compute (clear in nano photonics but likely in other approaches too) 

5. Compute requirements generally will significantly decrease (for a given capability)
